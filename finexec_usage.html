Finexec: Usage

Finexec | Documentation | Category | Usage | Style

Use the Finexec scripts and definitions to solve problems and develop your own defitions and projects. The scripts and definitions can be copied, modified as required for a project.

Overview

The JAL addons require to be installed.
The Finexec scripts are installed in the folder:~addons/finance/finexec
Individual scripts can then be used for a task or project.
Review the documentation for a script and definitions.

Loading Finexec Scripts

Generally Finexec scripts are loaded into the base locale.
eg: load '~addons/finance/finexec/basicfinance/simple.ijs'
Check the script scource for details.
eg: open '~addons/finance/finexec/basicfinance/simple.ijs'

If required the script can be loaded to another locale to aviod naming conflicts.
eg: load_simple_ '~addons/finance/finexec/basicfinance/simple.ijs'

Any dependencies for a script are listed and loaded.

Search Finexec

The Finexec scripts can be searched for keywords, phrases and text to find material for a problem or project. Use the search in files facility that is part of the J installation.
Search in the folder: ~addons/finance/finexec
