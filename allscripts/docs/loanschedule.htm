Loan schedule:
Using the Fold conjuctions F: and F.
No rounding.
Loan repayment are at the end of period.
Process:
(1) Set initial values for the loan schedule calculation.
(2) rub setv ’’ each time a calculation is required to
reset the initial values for the loan schedule.
(3) Run required definition:
      lshd ’’
      lshd_a ’’
