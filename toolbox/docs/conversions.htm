isscalar (monad)

Type: tacit
Test if scalar.
   1 = is a scalar
   0 = scalar

Syntax

toscalar(Z)
Z = item to be tested:

Example

   isscalar (,12)
1
   isscalar (12)
0
   isscalar (,'A')
1
