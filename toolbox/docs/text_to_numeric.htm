Definitions to convert text data to numeric data.

Definitions

D data, data_table, datas

data (monad)

Type: tacit
Create numeric data from a text noun.
Negative numbers to be in the format using underscore: _10 _12.1

Syntax

data(T)
T = numeric text data as a list or array

Example

   data('12 12.1 0.1')
12 12.1 0.1
   data('12 12.1 0.1 _10.1')
12 12.1 0.1 _10.1
 T =: 4 5 $ '1 2 3 4'
   data(T)
1 2 3 41 2 3 41 2 3

data_table (monad)

Type: tacit
Create numeric data from a text noun .
Negative numbers can be in the format -12.0 -10.123 and underscore: _10 _12.1.

Syntax

data1_table(T)
T = numeric text data created by  0 : 0

Example

 data_table DATA_TABLE
  $ DATA_TABLE
  320
  $ data_table DATA_TABLE
 8 8

datas (monad)

Type: tacit
Create numeric data from a text noun.
Negative numbers can be in the format -12.0 -10.123 and underscore: _10 _12.1.

Syntax

data1(T)
T = numeric text data as a list or array

Example

   datas('12 12.1 -0.1')
12 12.1 0.1
   datas('12 12.1 -0.1 -23.234 _10.1 ')
12 12.1 0.1 _10.1
