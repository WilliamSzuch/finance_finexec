Finexec: Addon

Finexec Addon Category 
Documentation
Documentation is provided at three levels:
  1) category
  2) script
  3) definition
Terminology is provided at the category level.
Equations are provided at the script and definition level.

Scripts
The Finexec addon scripts are installed in the folder: dir '~addons/finance/finexec'
Scripts for a category are in sub-folders. eg: dir '~addons/finance/finexec/basicfinance'
Individual scripts can be loaded for a task or project, eg: load '~addons/finance/fp/basicfinance/simple.ijs'

Definitions
Definitions can have tacit or explicit forms.
Examples are used to show the application of a definition. 
Definitions may also have several other forms: monadic and dyadic.

Equations
Equations are provided for scripts and definitions.
The equations are held by category, eg: dir '~addons/finance/finnexec/basicfinance/eqs'

Tutorials
Tutorials show the use of definitions to solve problems.
The tutorials are held by category, eg: dir '~addons/finance/finexec/basicfinance/tutes'
   
In the Qt or JHS session run: 
   finexec_tutorials ''
to list and run tutorials.

For an introduction to using tutorials run: 
   introduction_tut ''
