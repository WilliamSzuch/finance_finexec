Finexec: Addon

Finexec Addon Category Usage

Documentation

Finexec addon documentation is provided at three levels:
  1) category
  2) script
  3) definition

1) Category
The category documentation provides an overview of the subject or task. 
The categories are installed in the folder: dir '~addons/finance/finexec'
Category scripts are in sub-folders. eg: dir '~addons/finance/finexec/basicfinance'
A category may have a terminology link that sets out the meaning and usage of terms in the definitions.
 
2) Script
The script doscumentation provides an overview of the definitions for a subject or task.  
Individual scripts can be loaded for a task or project, eg: load '~addons/finance/finexec/basicfinance/simple.ijs'
The script scource can also be viewed. eg: Script source: simple.ijs
A script may have equations for its definitions.


3) Definition
The documentation for a definition provides detailed information for its application.
Examples are used to show the application of a definition. 
Definitions can have tacit or explicit forms.
Definitions may also have several other forms: monadic and dyadic.


Tutorials

Tutorials show the use of definitions to solve problems.
The tutorials are held by category, eg: dir '~addons/finance/finexec/basicfinance/tutes'
   
In the Qt or JHS session run: 
   finexec_tutorials ''
to list and run tutorials.

For an introduction to using tutorials run: 
   introduction_tut ''

Search Finexec Addon

The Finexec addon scripts can be searched for keywords, phrases and text to find subjects and definitions for a problem or project. Use the Find in Files facility that is part of the J installation.
Search in the folder: ~addons/finance/finexec.
