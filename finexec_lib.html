Finexec: Library

Finexec | Library | Documentation | Usage | Style

The Finexec library consists of:
(a) J programming scripts that are grouped by category related to the subject or task.
(b) Tutorials that provide examples of using the scripts and definitions to solve problems.
(c) Calculators

Category

Actuarial Life Life table, life annuities and life insurance
Actuarial Non-Life Tools for actuarial non-life and tasks
Basic Finance Simple and compound interest rate definitions
Calculators Basic financial and actuarial calculators
Date Time Definitions for date and time calculations
Derivatives Definitions for derivative calculations
Investment Performance Measurement of investment performance
J Phrases J Phrases
J Statistical Package J Statistical Package (Keith Smillie)
Qx Tables Various mortality tables
R Stats J interface to the R Statistcal Package
Toolbox Tools for developing scripts and definition for a task or project

Tutorials (under development)

Tutorials are run in a J session and are in early stages of development.
In the Qt or JHS session run: 
    finexec_tutorials ''
to list and run tutorials.
For an introduction to using tutorials run: 
   introduction_tut ''

Calculators

Financial and actuarial calculators are available in a Qt session.
For a list of calculators: Calculators
